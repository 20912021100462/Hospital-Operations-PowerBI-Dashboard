# Hospital Operations & Patient Satisfaction Dashboard 🏥

## 🌟 Project Overview
A comprehensive Power BI dashboard designed to provide a 360-degree view of hospital performance. This project focuses on operational efficiency, patient satisfaction (Patient Voice), and resource management (Bed Capacity) using advanced data storytelling and UI/UX principles.

## 🚀 Key Features
- **App-like Experience:** Interactive Landing Page with custom navigation buttons for seamless transitions.
- **Executive Overview:** High-level KPIs tracking total revenue, admissions, and departmental performance.
- **Operations Analysis:** Detailed heatmaps and decomposition trees to identify bottlenecks in hospital departments.
- **Capacity Management:** Real-time tracking of Bed Occupancy Rates to optimize patient flow.
- **Patient Satisfaction:** Analyzing "Patient Voice" data to improve quality of care and service delivery.

## 🛠️ Technical Stack
- **Tool:** Power BI Desktop
- **Data Modeling:** Star Schema (Fact and Dimension tables)
- **DAX:** Advanced measures for dynamic KPIs and conditional formatting.
- **UI/UX:** Glassmorphism design, custom icons, and interactive slicers.
